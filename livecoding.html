What is Java?

Java is an object-oriented programming language that was developed by james Gosling at Sun Microsystems, which is now acquired by Oracle, In today's time Java is one of the most diverse languages Java programming language is high-level object-oriented programming language designed to have a few implementation dependence as possible.

Key features of Java programming language:

Here are the important featues of the Java language

  1. Java programming language is designed to build object-oriented applications.
  2. Java is a multithreaded language with automatic memory management.
  3. In java, there is a feature called WORA i.e, write once run anywhere after compilation.
  4. Java code csn be run on any compyting platform.
